HTML Tags

  1. header: The <header> tag defines the header section of the body.
  2. img: The <img> tag introduces an image element in the webpage.
  3. p: The <p> tag defines a paragraph section of the webpage.
  4. b: The <b> tag bolds the text enclosed by it.
  5. table: The <table> tag creates a table.
  6. footer: The <footer> tag defines the footer section of the body.
  7. main: The <main> tag defines the main section of the body.
  8. mark: The <mark> tag highlights or marks the text encloses by it.
  9. link: The <link> tag creates a link with a current document. It is mostly used for styling.
  10. a: The <a> tag creates a hyperlink, which is used to move from one page to another.
