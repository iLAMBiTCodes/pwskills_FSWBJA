Features in HTML5

Semantic Tags

Unline HTML4, HTML5 has new and more meaningful tags which we call semantic tags. Some of the semantic tags include <header>, <nav>, <main>, <article>, <section> and <footer>.

Audio and Video tags

With the addition of <audio> and <video> tags in HTML5, it has become easy to add audio and video to the websites without using external plugins.

Drag and Drop

With this feature implemented on the site, the user can drag and drop media into the website.

Canvas

With the help of canvas, we can draw graphics, can create objects, images and animations with the help of JavaScript.

Local Storage

This new features helps store data in the user's browser locally and the data can be accesssed with the help of JavaScript API. This feature is useful when building offline applications.
