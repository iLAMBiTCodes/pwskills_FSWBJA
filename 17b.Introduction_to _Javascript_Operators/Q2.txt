Operators in JavaScript are categorized below:

1.Arithmetic Operators: Addition(+), Subtraction(-), Multiplication(*), Division(/), Modulus(%) are arithmetic operators that perform basic mathemtical operations.
Example:
let sum = 10 + 3; //Addition
let difference = 6 - 7; //Subtraction
let product = 2 * 2; //Multiplication
let quotient = 10 / 5; //Division
let remainder = 9 % 4; //Modulus

2. Comparison Operators: Comparison operators helps us to compare values and also data types and return a boolean result.
Examples:
let isEqual = (i == j)
let isNotEqual = (w != z)
let isStrictlyEqual = (a === b);  // Strict equality
let isStrictlyNotEqual = (x !== y); // Strict inequality
let isGreaterThan = (m > n); // Greater than
let isLessThanOrEqual = (p <= q); // Less than or equal to

3.Logical Operators: Perform logical operations on boolean values.
Examples:
let andResult = (true && false); // Logical AND(&&)
let orResult = (true || false); // Logical OR(||)
let notResult = !true; // Logical NOT(!)

4.Assignment Operators: Assign values to variables.
Examples:
let a = 2; // Assignment
a += 5; // Addition assignment (a = a + 5)
let b = 3;
y *= 9; // Multiplication assignment (b = b * 9)
let x = 54;
x /= 9; //Division assignment (x = x / 9)

5.Unary Operators: Perform/Operate on a single operand
Examples:
let a = 2;
a++; // Increment (a = a + 1)
let b = 1;
b--; // Decrement (b = b - 1)

6. Conditional/Ternary Operators: Providesa a way to express conditional statement.
Example:
let isLoggedIn = true;
let loggedState = isLoggedIn ? "Go to Dashboard" : "Login First";

7.Bitwise Operators: Perform operators at the bit level.
Examples:
let bitwiseAND = 1 & 3; // Bitwise AND
let bitwiseOR = 5 | 2; // Bitwise OR
let bitwiseXOR = 6 ^ 3; // Bitwise XOR
let bitwiseShift = 8 << 1; // Bitwise left shift
